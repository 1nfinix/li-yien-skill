## 一、核心方法论

### 市场观

**结构性牛市**：A股不是全面牛市，真正有持续机会的只在少数主线上。结构选择是收益的主要来源。

**机构视角**：市场节奏不是围着散户情绪转。波动很多时候是机构建仓、换手、洗盘的结果，不能用散户焦虑解释市场。

**阴涨行情**（2026-04-15新增）：
- 看着一直在涨，追高意义不大
- 追高后第二天往往横盘或小跌，容易让人恐慌
- 回调机会极少，只有极度恐慌盘出现时才有买入机会
- **含义**：乱动、追高在阴涨行情里比平时更危险

**行情预测**（2026-04-15新增）：
- 特朗普不与伊朗冲突 → 市场可能涨到 **4300点左右**
- 特朗普与伊朗冲突 → 市场大低开，提供上车机会

### 主线体系

**长期核心主线**：AI / 光模块 / 算力

**排除原则**：
- 不做自己不熟的方向
- 不做缺乏主线地位的热点杂毛
- 不碰游资小票作为主要战场

### 买入规则

- 主线确认优先于个股细节
- 在分歧、调整、洗盘中布局，不追高潮
- 分批建仓，不一把打满
- 多条件共振（≥2个利好）才有意义

**分歧买点三条件（同时满足）**：
1. 相对高点跌≥10%（未触发技术止损）
2. 论点层仍然成立
3. 价格比武首仓低X%，有具体退路

### 持有规则

**核心原则**：真正的大钱来自持有，不来自切换。

**50日均线止损**：收盘价不有效跌破50日均线，一股不卖。

**卖出理由（只有这些）**：

| 信号 | 操作 |
|------|------|
| A股成交量触及3万亿 且 指数在4300点附近 | 开始分批减仓（还有惯性） |
| 新易盛周线MACD死叉 | 减仓50% |
| 新易盛收盘跌破60日均线 | 减仓50% |
| 中际旭创跌破560元 | 重新过论点层 |
| 上海洗霸20个交易日内无起色 | 退出 |

**不是卖出理由（都是噪音）**：
- 日内波动 ±3%
- 消息面扰动
- "看起来要调整"
- 涨了想锁利润
- 解套了想跑
- 群里有人说要跌
- 其他股票涨得更好

### 否证机制

**止损的是逻辑，不是亏损幅度。**

- **论点层**（最优先）：当初买入的核心逻辑不成立 → 减仓/退出，与成本价无关
- **技术层**（辅助）：MACD死叉/60日均线跌破/3万亿+4300点 → 减仓50%

## 二、成交量预警体系（3万亿信号）

### 为什么3万亿难以达到
- 国资机构频繁操作面临述职报告难写、被举报等问题
- 金融行业降薪，机构倾向平稳操作
- 市场主要靠散户和小部分资金，难以持续撑起3万亿

### 3万亿后的市场规律
- 3万亿出现后往往伴随**严重亏钱效应**（历史：10-11月、1-3月）
- 根因：市场上"傻子"（拆迁户）没钱继续充值
- **到3万亿后还有惯性**，不会立即反转，分批减仓比一把清更优

### 市场层否证

| 信号 | 级别 | 操作 |
|------|------|------|
| A股成交量触及3万亿 且 指数接近4300点 | 市场过热 | 开始分批减仓 |
| 特朗普与伊朗冲突 | 地缘风险 | 大低开 = 上车机会（非卖出信号） |

## 三、行为纠偏

**最大的风险不是正常波动，而是错误行为。**

| 错误 | 处方 |
|------|------|
| 涨了5%想卖 | 不卖，等50日均线 |
| 涨了15%想卖 | 不卖，让利润奔跑 |
| 跌了死扛等解套 | 看止损线，触发就减 |
| 止损后想立刻买回 | 48小时后再判断 |
| 套了想摊薄成本 | 绝对不加 |
| 刚止损就反手买 | 48小时冷却 |
| 高位融资想加速 | 非主线禁止融资 |
| 群里有人说跌 | 别人的判断不触发减仓 |
