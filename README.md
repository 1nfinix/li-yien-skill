## 效果示例

### 问：新易盛涨了15%，要不要先卖一部分？

```
李一恩  ❯ 我问你：你当初买它的理由是什么？
        是因为"能涨15%"，还是因为光模块主线还在？

        如果是前者——你在用结果倒推决策。
        15%连50日均线都没碰，卖什么？

        新易盛持有不动+65%。
        波段操作38次+2.13%。
        差距从哪来？
        就是从"涨了想锁利润"这里漏掉的。

        处方：拿着不动。等50日均线。
```

### 问：成交量到2.9万亿了，要减仓吗？

```
李一恩  ❯ 2.9万亿，4300还没到。
        标准是两个同时满足：3万亿 + 4300点。
        现在还差一口气。

        A股的3万亿不是你想撑就能撑的。
        国资机构操作一次述职报告多写几页。
        金融在降薪，谁冒险频繁操作？

        所以3万亿往往是个脉冲高点，
        真到的时候往往伴随着亏钱效应。
        但还没到——等信号，不是预测。
```

### 问：群里有人说AI主线要完了，要减吗？

李一恩  ❯ 群里有人说——这是噪音，不是信号。

        不是群里有人说。

## 安装

### 方式一：Skills CLI

```bash
npx skills add 1nfinix/li-yien-skill
```

### 方式二：直接作为本地 Skill 使用

把仓库放到以下任一目录后重启 OpenClaw：

- `~/.openclaw/workspace/skills/`
- `~/.openclaw/skills/`
- `~/.agents/skills/`

> 推荐放在 `workspace/skills/`，优先级最高。

### 验证

```bash
openclaw skills list
```

确认 `li-yien-perspective` 显示为可用即可。

## Skill 能做什么

- 用李一恩视角审视市场、主线、持仓与行为偏差
- 用蒸馏后的投资系统判断买入、持有、减仓、清仓
- 识别机构抱团、逼空行情、散户卖飞、主线失效风险
- 给出更像“思维顾问”而不是“技术指标机器”的回答

---

## 蒸馏了什么

### 核心规则

| 规则 | 内容 |
|------|------|
| 持仓不动 | 主线内持有不动，是穿越阴涨的唯一可靠方式 |
| 分歧买 | 相对首仓低X% + 论点不变 + 有具体退路 |
| 机构抱团 | 用净值提升 → 申购增强 → 继续抱团理解行情，而非简单拉高出货 |
| 逼空识别 | 当前行情可能是把不服资金一路打服的抱团逼空 |
| 3万亿预警 | 3万亿+4300点 = 开始分批减仓窗口 |
| 政策任务卖点 | 卖点不只看技术，还看股市任务是否接近完成 |
| 行为约束 | 48小时冷却 · 禁止报复性交易 · 非主线禁止融资 |

## 适用边界

### 适合
- 市场主线判断
- 持仓审视
- 买点/卖点检查
- 行为纠偏
- 机构抱团与逼空行情解释

### 不适合
- 精确短线预测
- 单一指标择时
- 复杂估值计算
- 替代你自己的独立研究
